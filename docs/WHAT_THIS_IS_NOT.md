# What This Is Not

Direct boundary list for readers who move too fast.

This skeleton is not:

- A security guarantee
- A compliance product
- A production framework
- An autonomous agent launcher
- A trading system
- An open source version of a private operating system
- A replacement for human review
- Proof that agents are safe
- Permission for agents to act externally

## Preferred wording

This repo explores a small proof pattern.
It does not make agents safe.
It makes agent actions easier to inspect.
